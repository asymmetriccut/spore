# spore
Spore a stochastic generative sequencer for Monome Norns 
something is always becoming

A spore does not know what it will become. Cast into the world by wind or chance, it carries within it only potential — the quiet possibility of an organism not yet imagined.
Spore is a sequencer built around that same mystery.
You offer it seeds: a scale, a density, a tendency. Then you let go. From those first gestures, patterns germinate — branching, drifting, returning to themselves in ways you did not plan and cannot fully predict. Each session is a growth. Each performance, a new organism.
There is no loop here that repeats forever unchanged. There is only the ongoing process of becoming.

Spore is for those who find more music in a forest than in a metronome.
